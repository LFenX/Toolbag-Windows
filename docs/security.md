# 安全边界

Toolbag 默认普通权限启动，不以管理员身份常驻运行。所有系统能力都必须通过专用 Rust command 暴露，前端不得直接执行 shell。

## 默认策略

- 无遥测、无第三方崩溃上报。
- 日志只写本机，用户主动导出后才分享。
- 设置保存在本机 SQLite。
- 高风险工具必须声明 `riskLevel` 和 `requiresElevation`。

## 禁止事项

- 不提供通用任意命令执行器。
- 不把 updater 私钥、证书、token 或真实个人路径提交到仓库。
- 不让普通 UI 操作隐式提权。
- 不在前端绕过 `src/shared/tauri/commands.ts` 直接调用 IPC。

## 未来高权限工具

涉及注册表、服务、hosts、系统目录写入等能力时，按工具单独实现：

- 明确展示风险。
- 只对该操作做提权。
- 记录本地运行结果。
- 为失败路径提供可读错误信息。
